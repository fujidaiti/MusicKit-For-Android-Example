public static interface

MediaPlayerController.Listener

com.apple.android.music.playback.controller.MediaPlayerController.Listener

Class Overview

Interface defining callbacks that a MediaPlayerController can call on registered listeners. These methods will be called on the application's main thread, or the android.os.Looper specified at construction time.
